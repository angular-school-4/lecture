Angular.CLI

  • generate a new application 
                                
                            ng new my-app
  • start or build the application e.g. component 
                                
                            ng serve
                            ng build
  • create a new structural element 
                                
                            ng generate {type} {name}
  • run tests or check for lint violations 
                                
                            ng test
                            ng lint
                            ng lint --fix

Bootstrapping

  • loading and running applications,
  • provides control over the moment of launching the application,
  • allows to perform actions after launching the application,
  • allows to react when an error occurs during application startup.

Bootstrapping

main.ts

                    
                    if (environment.production) {
                        enableProdMode();
                    }

                    platformBrowserDynamic()
                        .bootstrapModule(AppModule)
                        .then(()=> console.log('App started!'))
                        .catch((err) => console.error(err));
